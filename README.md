# Flor
### Machine learning for the Great Outdoors

## Description
Flor is a machine learning app that utilizes image recognition to identify wildflowers.

## Technology
* React Native
* Expo
* TensorFlow
* Vize.ai
* Knex
* Node
* Express
* PostgreSQL
* Heroku
